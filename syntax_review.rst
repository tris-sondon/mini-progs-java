Java Syntax Review
========================

Two dimensional arrays
-----------------------

Declaration and allocation:

.. code-block:: java

   double[][] a = new double[N][M];

This will also initialize the array to zero.
If it is boolean, it will be initialized to false.

Compile-time initialization:

.. code-block:: java

   int[][] M = {
           {3, 4, 2},
           {1, 2, 8},
           {2, 3, 0},
           {7, 9, 1}
    };
